Range Addition II

题目：https://leetcode.com/problems/range-addition-ii/description/

题目大意

输入m，n生成m*n的矩阵M，默认用0填充
输入非负二维数组ops，对ops中的任意项[a,b]，凡是满足0 <= i < a 和 0 <= j < b的M[i][j]，M[i][j]加1，将ops视为对M的运算
当ops中的项执行完毕之后，统计M中最大元素出现的次数

思路

因为ops中元素是非负数组，+1都是M[0][0]开始，那么只要找到最小的操作项，就可以找到最大值

Construct the Rectangle

题目：https://leetcode.com/problems/construct-the-rectangle/description/

题目大意

给定一个数n，找出一对数L，W，满足以下条件

  1. n = L * W
  2. L >= W
  3. L和W的差别尽可能小

思路

两数差异最小时两数相等，转换考虑为L平方=n，同时L和W必然为n的约数，问题即转换为求n的约数中积最小的，且最大约数不超过n/2

Trapping Rain Water

题目：https://leetcode.com/problems/trapping-rain-water/description/

题目大意

Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it is able to trap after raining.

给定n个非负整数组成的宽度为1的柱状图，计算凹陷里面能够盛多少水

思路

上面的柱状图可以转换为二维数组，而根据木桶理论，能盛多少水是由最短边决定的，上面的问题转换为分割数组，按照数组内的非边界的最大值去分割，直到不可分割为止，最后计算数组可盛水的和即为整个图能盛水的量
